### Added

- Initial public release of the Rust `todui` CLI and full-screen TUI.
- SQLite-backed sessions, immutable revisions, markdown export, repo-aware todo metadata, and Pomodoro support.
- npm packaging for `@roberttlange/todui` with prebuilt macOS/Linux binaries plus a manual GitHub Actions release workflow.
